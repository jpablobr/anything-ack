Emacs Anything ack(1)

Functions:

M-x anything-ack

Installation:

In your emacs config:

(add-to-list 'load-path "~/.emacs.d/load/path/anything-ack.el")
(require 'anything-ack)

There is no need to setup load-path with add-to-list if you copy anything-ack.el to load-path directories.

Requirements

You will need to install anything-mode. See documentation for how to install.
